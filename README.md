# YiYue
## 简介
一乐是一款开源Android在线音乐播放器，目前支持以下功能：
* 在线与离线音乐播放
* 歌词显示，自动搜取歌词
* 磁带模式
* 定时停播、音效调节、闹钟、音效调节等辅助功能
